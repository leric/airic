# ls

Use ls to list entries in a directory and orient yourself in the workspace.

## When to use

- At the start of exploration when you do not know where relevant files live.
- To confirm a path exists before read or edit.

## Patterns

- ls at the workspace root, then drill into promising directories.
- Combine with find or grep when the directory is large.

## Avoid

- Listing every directory recursively by hand; prefer find for broad file discovery.
